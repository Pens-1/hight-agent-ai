# 画面設計（テキストワイヤーフレーム初版）

## 1. メインチャット画面
- ヘッダー
  - タイトル: hight-agent-ai
  - 設定ボタン（RAG トグルのみ表示、Web検索は未実装のため非表示または無効）
- メッセージリスト
  - ユーザー/アシスタント気泡
  - アップロード画像のサムネイル
  - 解答のMarkdownレンダリング（数式対応）
- 入力欄
  - テキスト入力
  - 画像アップロード（ドラッグ&ドロップ/ボタン）
  - 送信ボタン

## 2. 資料管理画面
- ツールバー
  - ファイルアップロードボタン
  - フィルタ（科目 / ステータス）
- テーブル
  - filename / subject / status / created_at / actions(delete)
  - ステータスはバッジ表示

## 3. 画面遷移
- ルート: /chat → メインチャット
- /documents → 資料管理
- ヘッダーから相互遷移

## 4. テーマ
- ダークテーマをデフォルト（Tailwind + prefers-color-scheme にも追従）
